# 权限解释技能

## 用途

当 Claude Code 弹出权限提示，而你想要更详细的解释时，调用此技能。

## 使用方法

输入 `/explain-permission` 即可。

## 我会做什么

1. 查看当前对话中最近需要权限的操作
2. 用通俗中文解释：
   - 这个操作在做什么（非程序员也能懂）
   - 风险等级和具体原因
   - 如果批准，最好的结果和最坏的结果
   - 如果不批准，会不会影响你的任务
3. 给出建议

## 风险等级说明

| 等级 | 含义 | 建议 |
|------|------|------|
| 🟢 低风险 | 只读操作，如查看文件、搜索 | 通常安全 |
| 🟡 中风险 | 修改文件内容 | 确认路径和内容正确后批准 |
| 🟠 高风险 | 删除文件、权限变更 | 仔细检查后再决定 |
| 🔴 严重 | 系统级破坏性操作 | 强烈建议拒绝 |

## 注意

- 只读操作一旦你允许"始终允许"，后续同类操作不再提示
- 涉及修改内容的操作每次都会提示，不会自动放行
